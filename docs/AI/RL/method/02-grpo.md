# 02 | GRPO (Group Relative Policy Optimization)

## 概述

GRPO 是 DeepSeek 提出的改进版 PPO，通过组内相对比较替代价值函数（Value Function），消除了对 Critic 模型的依赖，显著降低 RLHF 的训练成本。
