# 01 | PPO (Proximal Policy Optimization)

## 概述

PPO 是 OpenAI 提出的策略梯度算法，通过 clip 机制限制策略更新幅度，兼顾训练稳定性和样本效率。在 RLHF 中被广泛用于对齐大语言模型。
