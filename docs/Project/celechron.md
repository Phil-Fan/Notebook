# Celechron

!!! note "项目简介"
    - 项目名称：Celechron 的功能开发与维护
    - 一句话简介：使用 Dart+Flutter 开发的一款时间管理器，具有待办事项、时间统计、DDL 抓取等功能
    - 目的：了解 IOS 开发流程，学习 Flutter 框架，学习多平台开发
    - 相关链接：
    - 项目进程：进行中
    - 项目技术栈：Dart、Flutter、IOS、Android
    - 项目难点：时间统计、时间报告
